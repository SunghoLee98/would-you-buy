# 테스트 리포트 - 살래말래 (Would You Buy) 프로젝트

**프로젝트**: 살래말래 - 일일 주식 예측 투표 플랫폼
**테스트 일자**: 2025년 9월 23일
**테스트 대상**: 투표 대시보드 기능 (Issue #2)
**테스트 환경**:
- Frontend: React + TypeScript (localhost:5001)
- Backend: Spring Boot + Kotlin (localhost:7070)
- Database: PostgreSQL (localhost:5432)
